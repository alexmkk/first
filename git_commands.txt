запустить через контекстное меню в папке Git Bach Here

- gitk --all& (открыть окно управления версиями)
- ctrl + insert (копировать код версии)
- git checkout [shift+insert]

- git status - проверка состояний
- git diff - просмотр изменений

- git diff --staged - эта команда сравнивает ваши индексированные изменения с последним коммитом:

Если новый файл, то необходимо его добавить для отслеживание командой 
- git add Readme.txt
- git add . - добавить все новые

- git clone [url] (скачать проект с github)
- git commit -m -a "комментарий" (подготовить к заливке)

- git pull (скачать изменения)
- git push (залить изменения)
